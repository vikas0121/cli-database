# CLI DB


cli based json database

## Features

- add a json doc
- find a single doc by id
- findall docs
- delete a single doc by id

## Tech

cli-db uses a number of open source packages to work properly:

- "boxen": "^5.0.1",
- "chalk": "^4.1.2",
- "commander": "^8.1.0",
- "easy-json-database": "^1.5.0",
- "inquirer": "^8.1.2",
- "nanoid": "^3.1.25",
- "yargs": "^13.2.4"

## Installation

cli-db requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and devDependencies and start the server.

```sh
cd cli-database
npm i 
node bin/index.js
```

For install...

```sh
cd cli-database
sudo npm install -g
run
```
